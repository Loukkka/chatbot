📦 Guide d'intégration du Chatbot

Ajoutez le chatbot sur n'importe quel site web en copiant un simple bout de code.

1. Intégration simple (recommandée)

Collez ce code juste avant la balise </body> de votre site :

<!-- Chatbot IA --> <script> window.CHATBOT_CONFIG = { server: "https://VOTRE-SERVEUR.com" }; </script> <script src="https://VOTRE-SERVEUR.com/widget.js"></script>
💡 Remplacez https://VOTRE-SERVEUR.com par l'URL réelle de votre serveur chatbot.

2. Intégration avec personnalisation

Vous pouvez personnaliser les couleurs, le nom, etc. directement dans le code d'intégration :

<!-- Chatbot IA — Personnalisé --> <script> window.CHATBOT_CONFIG = { server: "https://VOTRE-SERVEUR.com", color: "#4B6BFB", // Couleur principale name: "Assistant", // Nom affiché welcome: "Bonjour !", // Message de bienvenue logo: "", // URL du logo (optionnel) position: "right", // "right" ou "left" placeholder: "Posez votre question..." }; </script> <script src="https://VOTRE-SERVEUR.com/widget.js"></script>

3. Options de configuration

Option Type Défaut Description
serverstringauto-détectéURL du serveur chatbot
colorstring#4B6BFBCouleur principale (hex)
namestringAssistantNom du chatbot
logostring""URL d'une image logo
welcomestringBonjour !Message de bienvenue
placeholderstringPosez votre question...Placeholder du champ de saisie
positionstringrightPosition du widget (right/left)
leadDelaynumber3Nb messages avant formulaire lead
leadKeywordsarray[...]Mots-clés déclencheurs du formulaire
